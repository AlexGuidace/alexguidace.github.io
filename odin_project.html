The Odin Project

The Odin Project is a website and community for those self-taught individuals looking to learn how to be professional web developers. It contains a collection of vetted learning resources from around the internet, compiled into learning modules--in a logical order. First you read (or watch) on a topic, then you build projects on that topic.

Those projects vary from the foundations of web development: HTML, CSS, and JavaScript, all the way up to more advanced technologies like React and NodeJS. I am going through this curriculum in order to hone my web development skills after having worked with web technologies in a professional capacity: TypeScript, React, and NestJS.
